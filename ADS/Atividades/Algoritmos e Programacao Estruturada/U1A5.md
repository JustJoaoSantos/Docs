# Desafio
- Uma pizzaria do bairro onde mora o procurou, pois costaria de automatizar seu caixa;
- implementar um calculo simples, em que, dado o valor total da conta de uma mesa, o programa divide esse valor pela quantidade de integrantes dessa mesa, e calcula o desconto concedido
- o programa deve receber como dados o valor da conta, a quantidade de pessoas e o percentual de desconto.
- com os dados no programa devera ser feito o calculo do valor total da conta com o desconto e o valor que cada pessoa devera pagar

# Resolucao
- Resolucao 
```c 
#include <stdio.h>

void main() {
float valor_bruto = 1000;
int qtd_pessoa = 10;
float desconto = 10;

float valor_liquido = valor_bruto - (desconto * (valor_bruto / 100));

printf("valor bruto: %f", valor_bruto);
printf("Valor liquido: %f", valor_liquido);
printf("Valor por pessoa: %.2f", valor_liquido / qtd_pessoa);
}
```
